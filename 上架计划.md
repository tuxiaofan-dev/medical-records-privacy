## 🔧 技术清单

### iOS 构建命令
```bash
cd flutter_app

# 清理
flutter clean

# 获取依赖
flutter pub get

# 构建 iOS
flutter build ios --release

# 在 Xcode 中打开
open ios/Runner.xcworkspace

# Archive -> Upload to App Store Connect
```

### Android 构建命令
```bash
cd flutter_app

# 清理
flutter clean

# 获取依赖
flutter pub get

# 构建 APK
flutter build apk --release

# 构建 App Bundle（推荐）
flutter build appbundle --release
```
